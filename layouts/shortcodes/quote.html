This is used inline. 2. quote_name: channel names, TV network name, brand/company names, author names, band names, singer names, actor names, character names, protocol names - first letter always capitalised 3. quote_title: albums, movies, televisions, radio shows, books, periodicals, databases, websites, service names, software names - in old print: underline - in English: italicized - Asian romanisation: « … ‹ … › … » - Chinese: 《…〈…〉…》 - Japanese: 《…〈…〉…》 4. quote_work: episode, articles, essays, chapters, poems, songs, speeches, mottos, ordinary quotations, release tags/titles/names - English: “ … ‘ … ’ … ” - Chinese: 「…『…』…」 - Japanese: 「…『…』…」 - South Korean (usually for vertical only): 「…『…』…」 */ -}} {{- $type := .Get "type" -}} {{- $lang := .Get "lang" -}} {{- $content := .Inner -}} {{- if eq $type "name" -}} {{ $content }} {{- else if eq $type "title" -}} {{ $content }} {{- else if eq $type "work" -}} {{ $content }} {{- end -}}
